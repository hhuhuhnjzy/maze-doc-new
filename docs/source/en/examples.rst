示例索引
========

以下示例覆盖从基础对话到复杂工作流编排的常见场景。每个示例均提供 MaClient 与 LanggraphClient 两种实现，方便对照学习。

使用说明
--------

- 按照 `Install <installation>`_ 章节准备环境。
- 复制示例代码到本地，替换为你的 API Key 或服务凭据。
- 根据需求选择 MaClient 或 LanggraphClient 版本，或结合两者混合部署。

对话问答
--------

**目标**：构建一个面向终端用户的问答机器人，支持上下文追踪。

- *MaClient*：展示如何使用 `WorkflowBuilder` 定义输入消息、响应节点与记忆模块。
- *LanggraphClient*：使用图形化节点描述对话状态机，并配置回溯策略。

工具调用
--------

**目标**：调用外部 REST API 获取实时数据（例如天气或股票）。

- *MaClient*：通过工具节点封装 HTTP 请求，并处理异常/重试。
- *LanggraphClient*：在图中接入外部服务节点，演示数据转换与条件分支。

多代理协作
----------

**目标**：让规划代理与执行代理协同完成复杂任务。

- *MaClient*：定义主流程与子任务节点，展示并行与聚合模式。
- *LanggraphClient*：以图描述多代理之间的消息传递和角色切换。

批量处理
--------

**目标**：离线批量处理输入数据，生成摘要或分类结果。

- *MaClient*：演示批处理作业调度与进度追踪。
- *LanggraphClient*：构建批量管道，强调资源配额与回压控制。

进一步扩展
----------

每个示例可作为模板扩展成你的业务场景。建议将自定义示例保存在 `examples/` 子目录，并在此页面通过 `.. toctree::` 动态导入，确保 Read the Docs 自动收录。
